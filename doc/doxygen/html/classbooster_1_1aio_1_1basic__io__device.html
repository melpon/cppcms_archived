Detailed Description

This is a basic object that allows execution of asynchronous operations.

It represents a generic "select"able file descriptor or SOCKET on Windows platform.

It does following:

  • Connects an object with the event loop - io_service
  • Allows to connect and disconnect native file descriptor/socket object to it, and close it
  • Asynchronously poll the object for readability and writeability via io_service object and cancel such operations
  • Switch blocking and non-blocking mode of the descriptor.
