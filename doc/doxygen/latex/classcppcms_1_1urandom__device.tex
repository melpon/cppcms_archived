\subsection{Detailed Description}
High entropy random number generator. 

This is cryptographic random number generator that uses /dev/urandom on P\-O\-S\-I\-X platforms and Crypto\-A\-P\-I's Crypt\-Gen\-Random function under M\-S Windows
