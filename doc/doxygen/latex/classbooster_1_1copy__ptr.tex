\subsection{Detailed Description}
\subsubsection*{template$<$typename T$>$class booster\-::copy\-\_\-ptr$<$ T $>$}

a smart pointer similar to std\-::auto\-\_\-ptr but it copies underlying object on pointer copy instead of moving its ownership. 

Note\-: Underlying object has same constness as the pointer itself (not like in ordinary pointer).

Don't use it with polymorphic classes. Prefer \doxyref{clone\-\_\-ptr}{p.}{classbooster_1_1clone__ptr} instead.
